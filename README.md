# 💡 Calculadora Inteligente de Fermentação

![Preview do Projeto](./src/assets/app-fermentação.png)

## 📌 Sobre o projeto

Durante a rotina na padaria, enfrentávamos um problema constante: a fermentação nem sempre saía como o esperado. Mesmo seguindo uma quantidade “padrão” de fermento, o resultado variava bastante dependendo da temperatura e do tempo — o que impactava diretamente na qualidade dos pães.

Pensando nisso, desenvolvi uma aplicação web + mobile utilizando **React**, **HTML** e **Tailwind CSS**, com o objetivo de tornar esse processo mais previsível e eficiente.

---

## 🔧 Funcionalidades

- 🍞 Calcula a quantidade ideal de fermento com base na temperatura ambiente e no tempo de fermentação  
- 💡 Exibe dicas práticas para melhorar o resultado da massa  
- 📊 Armazena histórico dos cálculos para acompanhamento  
- 📱 Interface responsiva (web e mobile)
